이용안내

  • 음성통화/문자

    • 1)음성통화/문자 월 기본 제공량은 아래와 같습니다.
      • 무선(통신사번호 : 010, 011 등), 유선(지역번호 : 02, 031 등), 인터넷전화(070) 이용 시 사용량에 관계없이 무제한 무료 통화 가능
      • 그 외 부가통화 전국대표번호(15XX,16XX), 평생개인번호(050X), 주파수공용통신(013등) 및 영상통화의 경우 월 300분 별도제공량에서 차감(부가음성통화 1초당 1초, 영상통화 1초당 1.66초 차감)
      • 부가통화 이용시 부가통화 사업자가 정하는 바에 따라 별도의 정보이용료가 발생될 수 있음
      • 기본제공 메시지 적용대상은 SMS, LMS, MMS 이며 정보이용료 등은 별도 부가함
    • 2)월 기본 제공량을 모두 사용시 적용요율은 아래와 같습니다
      • 기본제공량 초과시 요율 : 음성(초) : 1.98원 / 영상(초): 3.3원 / SMS(건): 22원 / LMS(건): 33원 / MMS(건): 220원 / DATA(MB) : 22.53원
    • 3)스팸 등 상업적 목적으로 사용하는 경우 요금을 청구하거나 이용이 정지되며 해당 기준은 다음과 같습니다
      • 음성통화량이 일 600분을 초과하는 횟수가 월 중 3회가 넘는 경우(영상통화는 음성의 1.66배로 계산)
      • 음성통화량이 월 10,000분을 초과할 경우(영상통화는 음성의 1.66배로 계산)
      • TM/폰팅 등과 같이 음성/영상통화를 광고-상업적 목적으로 이용하는 경우(음성/영상통화 수신처가 월 1,000개 회선을 초과하는 경우에는 광고-상업적 목적으로 이용하는 것으로 간주함
      • 하루 150건 초과하는 메시지를 월 10회 초과하여 보내는 경우
      • 하루 500건을 초과하는 메시지를 보내는 경우
      • 한 달에 3,000개를 초과하는 번호로 메시지를 보내는 경우
      • 휴대폰 이외의 시스템을 통해 메시지를 발송하는 경우
      • 제 3자에게 휴대전화를 임대하는 등 이용약관을 위반하는 경우
    • 4)국제전화는 별도요율로 과금됩니다.

  • 데이터

    • 1)기본제공 데이터는 순수 데이터 통화용으로만 사용이 가능하며 기본제공량 초과 시, 속도제어 상태로 무제한 이용 가능 합니다
    • 2)테더링에 사용되는 데이터는 기본제공 데이터(월제공량 또는 일제공량)내에서 합산하여 차감되며, 기본제공 데이터 소진 후 속도제어시 테더링은 자동차단 됩니다.
      • 「LTE무제한11GB+(KT망)」,「LTE무제한11GB+(왓챠)(KT망)」 요금제의 경우 월 11GB+ 일2GB 제공량까지 테더링 이용가능
    • 3)데이터쉐어링에 사용되는 데이터는 기본제공 데이터(월제공량 또는 일제공량) 내에서 합산하여 차감 되며 기본제공 데이터 소진시 200Kbps 이하의 속도로 지속 이용 가능합니다.
      • 「LTE무제한11GB+(왓챠)(KT망)」 요금제의 경우 월 11GB +일2GB 제공량까지 쉐어링 이용가능
      • 「LTE무제한11GB+(KT망)」 요금제의 경우 월 10GB +일2GB 제공량까지 쉐어링 이용가능
